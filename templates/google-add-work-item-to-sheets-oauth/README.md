# Google - Add Work Item to Sheets (OAuth)

Adds Jira work items to a Google Sheet.

Use this template when you want a spreadsheet to be updated automatically whenever a Jira work item event happens.

The template adds a new row to the selected Google Sheet with the Jira work item key and summary. This gives teams a simple spreadsheet view of Jira activity without manually copying information from Jira.

This is useful for lightweight reporting, handover lists, request logs, external team tracking, or any process where a Google Sheet is easier to share than a Jira view.

This version uses Google OAuth, so it connects through an authorized Google account with access to the spreadsheet.
