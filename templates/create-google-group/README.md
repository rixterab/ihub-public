# Create Google Group

Creates a Google Group from a Jira request.

Use this template when users should be able to request a new Google Group through Jira instead of asking an administrator to create it manually.

The Jira request provides the group name and description. The template creates the Google Group, adds the selected users as members, and then updates the Jira ticket so the requester can see that the group is ready.

This is useful for standardizing group creation, reducing manual admin work, and making sure every new group request is tracked in Jira.
