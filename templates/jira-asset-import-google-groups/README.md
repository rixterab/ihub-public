# Jira Asset Import: Google Groups

Keeps a list of your Google Groups in Jira Assets.

Use this template when you want Jira to show which Google Groups exist in your organization, including each group's name, email address, Google ID, and number of direct members.

This is useful for keeping an overview of access groups, ownership groups, or mailing lists inside Jira Assets, so teams can search and report on them without opening Google Admin.

The template is set up to run on a schedule and refresh the information automatically.
