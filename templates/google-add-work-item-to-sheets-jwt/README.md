# Google - Add to Sheet (JWT)

Adds Jira work items to a Google Sheet.

Use this template when you want a spreadsheet to be updated automatically whenever a Jira work item event happens.

The template adds a new row to the selected Google Sheet with the Jira work item key and summary. This gives teams a simple spreadsheet view of new or changed Jira items without manually copying information from Jira.

This is useful for lightweight reporting, handover lists, request logs, external team tracking, or any process where a Google Sheet is easier to share than a Jira view.

This version uses Google JWT credentials, which is commonly used for service-account based access.
