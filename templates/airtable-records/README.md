# Airtable

Gets records from an Airtable table.

Use this template when you want another system or workflow to fetch rows from a specific Airtable base and table. You configure which Airtable base and table to read from, then the template returns the records from that table.

This is useful when Airtable is used as a lightweight database, planning sheet, inventory list, request tracker, or source list, and you want that information available in an integration without manually exporting it.

The template is started by a webhook, so records are fetched when another workflow or system asks for them.
