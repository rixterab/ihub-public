# AWS EC2 Repo S3

Gets an overview of key AWS resources.

Use this template when you want to collect information about EC2 instances, S3 buckets, and container repositories from an AWS account. It can help teams keep track of cloud resources without manually checking several AWS pages.

This is useful for asset inventory, audits, cleanup work, cost reviews, or giving support and operations teams a simple way to see what exists in AWS.

The template includes separate scheduled flows for EC2, S3, and repositories, so each resource type can be refreshed automatically.
